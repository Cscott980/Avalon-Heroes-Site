⚠️
Alpha Code — Distribution Policy
Access codes are provided for classroom use only. By requesting a code you agree to the following:
  • Do not sell the access code or charge any fee for access to the game.
  • Do not publicly post the code online, in forums, social media, or any public channel.
  • Distribute only to your own students who are participating in the classroom trial.
  • Codes may be revoked if misuse is detected.
If you have questions about permitted use, please contact us before sharing the code.
